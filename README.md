# 电视语音场景对接源码

#### 介绍
电视语音指令透传源码参考，动态对接夏杰语音，直接支持语音交互。
该对接方式不限定技能类型，只当对应app处于当前运行页面时，才接收相关指令。

如果App是直播、影视、音乐等，请用对应技能对接sdk。

#### 场景要求
1. 夏杰语音需要开启获取软件状态权限，可以安装AI智能管家辅助自动开启。
2. 该APP动态注册后，并处于当前运行界面，才能接收到指令。

#### 使用说明

1.  下载源码编译，也可以合入到自己工程里面。
